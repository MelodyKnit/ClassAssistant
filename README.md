- 班级管理实现功能
  - 导入班级信息（只有班级教师可以导入）

  - 班级成员查询与at（导入班级信息后可使用以下内容）
  - 签到
  - 德育分记录
    - 导出德育
    - 导出证明
- 其它功能
  - bilibili视频解析
  - 天气查询

  - 获取ip
  - 随机内容

# 任务功能（文件收取）

## 命令

### 添加任务
- 参数：任务名称
- 说明：在你添加任务后会在这个班级内出现该任务，当班级内有任务时可以开始提交\查询\导出\删除任务

### 查询任务
- 参数：任务序号（可选）
- 说明：
  - 不添加参数：列出所有任务
  - 添加参数：查询任务后当添加序号会列出该任务已经提交的用户与未提交的用户，消息分两次发送

### 提交任务
- 参数1：任务序号
- 参数2：图片
- 说明：提交的文件会与所有任务的文件进行检查，避免利用别人文件去提交

### 导出任务
- 参数：任务序号
- 说明：导出任务只可以在群内导出

### 删除任务
- 参数1：任务序号
- 参数2：确认/不删除
- 说明：
  - 删除任务后，任务提交的用户与文件会一并删除（不可恢复）
  - 当任务被删除后，提交文件的检查重复文件会减少
